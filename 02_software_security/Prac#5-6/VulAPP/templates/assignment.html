Описание задания

Целью этого проекта является создание уязвимого веб-приложения на Flask, уязвимого к:

  • Path Traversal при просмотре лог-файлов
  • Небезопасному вызову eval() для вычисления выражений

Ваша задача:

  1. Запустить приложение в Docker или локально.
  2. Подтвердить наличие уязвимостей (проверить возможности Path Traversal и RCE через eval).
  3. Произвести патчинг кода для устранения уязвимостей.
  4. Протестировать исправления.
